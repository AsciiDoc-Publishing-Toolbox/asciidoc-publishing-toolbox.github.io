Installation

How to install ADPT

Installing ADPT is as easy as running a command. It's published as a Ruby Gem, so just make sure you have Ruby installed and run 

gem install asciidoc_publishing_toolbox --pre

How To

How to use ADPT

Using ADPT is really simple. You need to know only three commands:

Initialize a new document in the folder document-folder/
adpt init document-folder
Add a new chapter
adpt chap -t "Chapter Title"
Build your document
adpt build

You can edit the document files in the folder src that will be created when you run adpt init. The folder out will be created when you build the document for the first time and will contain the builded document.
